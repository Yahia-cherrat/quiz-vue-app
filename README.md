# Quiz App

A simple quiz application built with Vue.js that allows users to test their knowledge on various topics.

## Features

- Interactive quiz interface
- Multiple-choice questions
- Display of results at the end of the quiz
- Responsive design

## Technologies Used

- Vue.js
- Vue Router
- HTML5
- TailwindCss
